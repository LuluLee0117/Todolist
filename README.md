# todofrontend

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Open Backend
```
node server.js
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
